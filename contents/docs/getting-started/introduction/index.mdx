你好，这里是 **Grtsinry43's Learning Notes** ！

流水不争先，争的是滔滔不绝，对于前端的学习，唯有不断积累和总结，才能不断进步。

这里是我个人的学习笔记，记录了我在前端学习过程中的一些心得体会和技术总结，希望能够帮助到你。

## 这个网站有什么

- **学习笔记**：记录了我在前端学习过程中的一些心得体会和技术总结。
- **技术文档**：记录了一些技术文档，方便查阅。
- **项目实战**：实际项目中遇到的问题和解决方案。
- **学习资源**：一些学习资源的整理和分享。

## 为什么要构建这个笔记

前端技术日新月异，学习的内容繁杂而琐碎，我希望通过这个笔记，将自己的学习过程记录下来，方便自己查阅，也希望能够帮助到其他人。

从不久以前，我开始写作，想要把自己的所学所得记录下来，而博客承载的内容毕竟有限，而且只有广度而缺少深入研究，

所以我决定构建这个笔记，记录下自己的学习过程，也希望能够帮助到其他人。

## 它是基于什么构建的

这个网站是基于 **AriaDocs** 模板构建的，它提供了一个现代、灵活和用户友好的基础，用于创建引人入胜的文档。

其由 Next.js 15、Tailwind CSS、Shadcn-UI、next-mdx-remote 等技术构建，保证了性能和灵活性。

- **Next.js 15** - 优化生产的强大 React 框架。
- **Tailwind CSS** - 为快速、干净的设计提供实用的样式。
- **Shadcn-UI** - 优雅、易用的组件，打造精致的外观。
- **next-mdx-remote** - 为动态、交互式 Markdown 内容提供 MDX 支持。
