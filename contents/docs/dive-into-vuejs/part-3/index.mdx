## 关于 v-model

`v-model` 本质就是一个语法糖，生成一个属性 `value` 和一个事件 `input`

默认情况它等效于：

```html
<Comp v-model="data" />

<Comp :value="data" @input="data=$event" />
```

开发者可以修改属性和事件的名字

```javascript
model: {
    prop: "number",
    event: "change"
}
```

这样就

```html
<Comp :number="data" @change="data=$event" />
```

<Note title="思考">

这里有点想法，就是 MVVM 框架的设计思路，React 是通过 onChange 事件，然后可以调用 set 方法，而 Vue 是通过自定义 emit 来达到相互的效果

</Note>
